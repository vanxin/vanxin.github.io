## 安装需求

   eclipse 3.6.0以上
   java 7


## 安装步骤（with update site）

   开启Eclipse
   进入“help”工具栏选择“Install New Software...”
   点击“Add...”在Add Site中，Name填入：Pydev and Pydev Extensions,在Location填入: http://pydev.org/updates
   等待获取，选择“Pydev”点击“Next”进入下一步，接受版权声明，等待安装成功，重启即可。
